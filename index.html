Cómo funciona

Este modelo separa muy bien lo público de lo privado.

1. BSS crea o activa el perfil del negocio.
2. El cliente personaliza su espacio público.
3. El perfil se usa para promoción y visibilidad.
4. La operación real queda fuera del área pública.

BSS Admin Generator

Uso:
1. Abre admin-generator.html en el navegador.
2. Llena los datos del negocio.
3. Presiona Generar.
4. Copia el bloque para data.js y pégalo dentro del array.
5. Descarga el perfil individual y súbelo al repo.
6. Haz Commit changes en GitHub.

Esto no publica automáticamente todavía.
Sirve para evitar errores y acelerar la creación de perfiles.
